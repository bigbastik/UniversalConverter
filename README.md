\# Universal File Converter PRO (Versione Portable)



Universal File Converter PRO è un convertitore di file con interfaccia grafica moderna che permette di convertire facilmente PDF, documenti Office e immagini.



Questa versione è \*\*portable (.exe)\*\*: non richiede installazione.



---



\## 🚀 Come usare il programma



1\. Scarica il file `.exe`

2\. Avvia il programma con doppio clic

3\. Clicca su \*\*"Seleziona File"\*\*

4\. Scegli il tipo di conversione desiderato

5\. I file convertiti verranno salvati nella stessa cartella degli originali



---



\## 🔄 Conversioni supportate



\* PDF → DOCX

\* DOCX → PDF \*(richiede Microsoft Office)\*

\* XLSX → PDF \*(richiede Microsoft Office)\*

\* PPTX → PDF \*(richiede Microsoft Office)\*

\* JPG / PNG → PDF (multipagina)

\* PDF → Immagini (una per pagina)

\* PNG → JPG

\* JPG → PNG



---



\## ⚙️ Requisiti di sistema



\* Windows 10 / 11

\* Microsoft Office (necessario solo per conversioni Office → PDF)

\* Poppler (necessario solo per PDF → Immagini)



---



\## 📦 Versione Portable



\* Nessuna installazione richiesta

\* Funziona offline

\* Interfaccia grafica semplice

\* Supporto multi-file



---



\## ⚠️ Note importanti



\* La conversione DOCX/XLSX/PPTX in PDF funziona solo se Microsoft Office è installato.

\* La conversione PDF → immagini richiede Poppler configurato nel sistema.

\* File PDF molto grandi possono richiedere più tempo per la conversione.



---

\* Python

\* pdf2docx

\* Pillow (PIL)

\* pdf2image

\* pywin32 (solo per Office → PDF)
